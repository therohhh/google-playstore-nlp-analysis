# Google Play Store Data Analysis and Sentiment Analysis

## Overview
This project analyzes the Google Play Store dataset to extract insights about apps, ratings, and user reviews. The project includes data preprocessing, exploratory data analysis, and sentiment analysis on user reviews.

## Objectives
- Clean and preprocess raw dataset
- Perform exploratory data analysis
- Apply sentiment analysis on app reviews
- Visualize insights using interactive plots

## Technologies Used
Python  
Pandas  
NumPy  
Matplotlib  
Plotly  
Scikit-learn  
Natural Language Processing (NLP)

## Project Workflow
1. Data Cleaning
2. Data Transformation
3. Sentiment Analysis
4. Data Visualization

## Key Insights
- Distribution of app ratings across categories
- Sentiment patterns in user reviews
- Trends in app popularity and installs
